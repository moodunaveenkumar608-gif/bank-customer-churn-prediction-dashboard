# 🏦 Bank Customer Churn Prediction Dashboard

This project predicts whether a bank customer is likely to leave the bank using machine learning.

https://bank-customer-churn-prediction-dashboard-bzt4hqmo7zwymfzq5awat.streamlit.app/

## Features
- Customer churn prediction
- Churn probability
- Dashboard and charts
- Prediction history
- Download CSV

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- Plotly
